Introduction¶

1.Project Objective:

The primary objective of this project is to predict housing prices using a dataset from Kaggle, specifically focusing on the application of Kernel Ridge Regression. The performance of this model will be compared with three other machine learning models: Linear Regression, Logistic Regression, and Support Vector Machine (SVM) and Decision Tree.

2.Dataset Description: I'm using a dataset from Zillow, which includes Austin area listings from August 2021. Link:https://www.kaggle.com/datasets/saritasanchez/zillow-listings-in-austin-area-from-aug-2021?select=location.csv The dataset contains 3 CSV files: description features.csv, location.csv, other features.csv, and pricing.csv, totaling 15172 records with 31 discernible features. It includes a column with the latest house prices. I've selected 8 key features for my model which are propertyTaxRate,yearBuilt,avgSchoolRating,zipcode,latitude,longitude,numofBathrooms,numofBedrooms,lotsizeSqFt,and homeType. The dataset will be split with 80% for training and 20% for testing. And for hyperparameter tuning,I will use tool called GridSearchCV from. GridSearchCV can systematically work through multiple combinations of parameter tunes, cross-validating as they go to determine which tune gives the best performance.

from sklearn.kernel_ridge import KernelRidge
from sklearn.model_selection import GridSearchCV
import matplotlib.pyplot as plt
import numpy as np
import time

# Define the Kernel Ridge Regression model
kr_model = KernelRidge()

# Define the parameter grid for hyperparameter tuning
# explores different combinations of the specified alpha and kernel parameters to find the best settings for your Kernel Ridge Regression model.
param_grid = {
    'alpha': [0.1, 1, 10],
    'kernel': ['linear', 'rbf', 'poly']
}

# Set up GridSearchCV
grid_search = GridSearchCV(kr_model, param_grid, cv=5, scoring='neg_mean_squared_error')

# Perform the grid search on the training data
timeForTraining_kernelRidge0 = time.time()

grid_search.fit(X_train, y_train)

# Retrieve the best model
best_kr_model = grid_search.best_estimator_

timeForTraining_kernelRidge1 = time.time()


from sklearn.metrics import mean_squared_error, r2_score
import time

# Predict on the test data

y_pred_kernelRidge = best_kr_model.predict(X_test)

timeForPredict_kernelRidge1 = time.time()
